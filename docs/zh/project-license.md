## 这意味着什么

- 用于任何用途，包括商业工作。
- 修改、分叉、重新分发。
- 如果重新分发，请保留源代码中的许可证声明。

## 基于 Codex

Open Interpreter 是 [Codex](https://developers.openai.com/codex) 的分支，并继承了其终端体验。沙箱模式、审批策略以及 `AGENTS.md` 等概念都来自其中。该项目保持对上游友好，同时保持供应商无关。

## 商标

Open Interpreter 是 Open Interpreter 社区的项目。Codex 名称和商标归 OpenAI 所有，未在本项目中获得许可。
