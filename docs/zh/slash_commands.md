---
title: 斜杠命令
description: 可在交互式编辑器中使用的命令。
---

在编辑器中输入 `/` 可打开命令选择器。列表会根据当前模式和已启用的功能标志进行调整。

## 模型与行为

| 命令 | 用途 |
| ------- | ------- |
| `/model` | 选择提供商、模型以及推理力度。 |
| `/fast` | 使用可用的最快服务层级。 |
| `/personality` | 选择交流风格。 |
| `/theme` | 更改语法高亮主题。 |
| `/status` | 显示模型、提供商、沙箱、批准情况、令牌使用以及会话状态。 |

## 权限与沙箱

| 命令 | 用途 |
| ------- | ------- |
| `/permissions` | 选择当前的权限姿态。 |
| `/approvals` | 在兼容构建中作为 permissions 的别名。 |
| `/sandbox-add-read-dir <path>` | 授予对额外目录的读取权限。 |
| `/setup-default-sandbox` | 在支持的环境中配置默认沙箱。 |

## 对话与会话

| 命令 | 用途 |
| ------- | ------- |
| `/new` | 开启新对话。 |
| `/resume` | 恢复之前的会话。 |
| `/fork` | 将当前会话分支为新会话。 |
| `/side` | 在分支中启动侧向对话。 |
| `/agent` | 切换或管理子代理线程。 |
| `/clear` | 清除可见的对话记录。 |
| `/compact` | 汇总旧上下文以释放空间。 |
| `/rename` | 重命名当前线程。 |
| `/exit`, `/quit` | 退出 TUI。 |

## 文件与代码

| 命令 | 用途 |
| ------- | ------- |
| `/init` | 创建包含项目指导的 `AGENTS.md`。 |
| `/mention` | 将文件添加到对话中。 |
| `/diff` | 显示当前工作树的差异。 |
| `/review` | 审查当前更改以发现 bug 和回归。 |
| `/copy` | 复制最新的助手响应。 |

## 集成

| 命令 | 用途 |
| ------- | ------- |
| `/mcp` | 显示已配置的 MCP 服务器和工具。 |
| `/skills` | 检查可用技能。 |
| `/plugins` | 在启用时浏览插件支持。 |
| `/apps` | 在启用时管理连接器/应用工具。 |
| `/memories` | 在启用时检查或配置记忆。 |
| `/hooks` | 审查并信任生命周期钩子。 |

## 后台工作

| 命令 | 用途 |
| ------- | ------- |
| `/ps` | 列出后台终端任务。 |
| `/stop` | 停止后台任务。 |

## 诊断与生命周期

| 命令 | 用途 |
| ------- | ------- |
| `/debug-config` | 打印已解析的配置及其来源。 |
| `/rollout` | 显示本地对话记录路径。 |
| `/title` | 配置终端标题行为。 |
| `/statusline` | 配置状态栏行为。 |
| `/experimental` | 在可用时切换实验性功能。 |
| `/update` | 管理独立 CLI 更新。 |
| `/logout` | 移除已保存的凭据。 |
| `/feedback` | 发送反馈及可选日志。 |
