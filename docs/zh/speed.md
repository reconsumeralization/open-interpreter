速度取决于模型选择、上下文大小和工具设置。

## 快速模式

在活动模型/提供商支持的情况下使用 `/fast`。对于持久默认设置：

```toml
[profiles.fast]
model_reasoning_effort = "low"
```

## 保持上下文紧凑

- 使用 `@` 仅提及相关文件。
- 使用 `/compact` 缩短冗长的会话。
- 将持久的仓库规则放在 `AGENTS.md` 中，而不是每个提示。
- 将可重复的流程放入技能中，以便仅在需要时加载。

## 工具

让构建和测试命令可以轻松从仓库根目录运行。并在 `AGENTS.md` 中记录它们。
