## Built on Codex

Open Interpreter is a fork of [Codex](https://developers.openai.com/codex)
and inherits its terminal experience. Concepts like sandbox modes,
approval policies, and `AGENTS.md` come from there. The project remains
upstream-friendly while staying provider agnostic.
