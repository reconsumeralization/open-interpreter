# Open Interpreter

<DocsTerminalHero />

<DocsAgentHandoff />

Open Interpreter 是一个运行在终端中的编码代理。打开一个项目，输入 `interpreter`，它会读取你的文件、编辑它们并代表你执行命令。

它基于 Codex 构建，并保持对提供商中立，因此你可以接入 OpenAI、Anthropic、本地模型或其他任何模型。

## 功能概述

- **就地编辑代码。** 提出修改请求并在提交前审查差异。
- **在明确的护栏下运行命令。** 沙箱决定哪些操作是可能的，审批模式决定何时先询问。
- **使用你自己的模型。** 选择适合任务的提供商和模型。随时使用 `/model` 切换。
- **恢复长时间会话。** 使用 `interpreter resume --last` 继续之前的工作。
- **保持 Codex 的使用体验。** 使用熟悉的 `exec`、会话、MCP、`AGENTS.md`、技能、钩子、权限和子代理，配合 Open Interpreter 的提供商无关模型选择器。

## 使用体验

```bash
$ cd my-project
$ interpreter
> add a /health endpoint that returns the build sha
```

Open Interpreter 会规划变更、编辑文件，并在超出当前沙箱范围时先询问确认。
